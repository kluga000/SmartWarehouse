# 📦 SmartWarehouse

Інформаційна система для автоматизації складського обліку, розроблена в межах навчального практикуму. Проєкт реалізує повний цикл керування товарними запасами з використанням сучасних підходів до розробки на C#.

## 🛠 Технології та архітектура
* **Мова програмування:** C# (.NET 8.0)
* **Графічний інтерфейс:** WPF (XAML)
* **Архітектурний патерн:** Repository Pattern для ізоляції логіки роботи з даними
* **Формат збереження даних:** JSON (локальні файли)

## 🚀 Ключові особливості (Результати рефакторингу)
* **Smart Upsert:** Система автоматично перевіряє артикул при додаванні. Якщо такий товар існує, кількість оновлюється автоматично, запобігаючи дублюванню.
* **Відновлення з архіву:** Реалізовано логіку «м'якого видалення», що дозволяє відновлювати товари з архіву при повторному введенні їхнього артикула.
* **Ієрархія категорій:** Підтримка деревовидної структури категорій для зручної класифікації складських запасів.

## 🧪 Тестування
Проєкт містить модульні тести (Unit Tests), які перевіряють критичну бізнес-логіку:
* Авторизація користувачів із різними рівнями доступу (Admin/User).
* Розрахунок повної назви категорії з урахуванням батьківських елементів.
* Коректність оновлення кількості товару при додаванні.

## 📂 Встановлення
1. **Для розробників:** Відкрийте файл `SmartWarehouse.sln` у Visual Studio 2022 та натисніть F5.
2. **Для користувачів:** У папці `publish` запустіть файл `setup.exe` для встановлення застосунку в систему.
